# java-virtual-world 
Event-driven 2D world simulation in Java with custom entities, scheduler, and architecture.

Key Features: 
Event-driven scheduler for animations & actions Cohesive class architecture (refactored from a legacy codebase) 
Custom entities: volcanoes, lava monsters, burnt terrain, trees, fairies, miners, etc. Image loading, sprite animation, and dynamic transformations WorldModel, VirtualWorld, ImageStore, and EventScheduler restructured for readability 

Language/techniques: 
Java 
OOP principles (interfaces, abstract classes, sub class, etc) 
Data structures (lists, arrays, hash maps, spatial grids, priority queue scheduler, Java Streams API)
Testing: manual simulation testing, debugging using logs and breakpoints
